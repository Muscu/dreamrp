VEHICLE FILES
   Using OpenIV or similar program, put the vehicle files (riot_hi.yft / riot.yft / riot.ytd) into:
            update>x64>dlcpacks>patchday8ng>dlc.rpf>x64>levels>gta5>vehicles.rpf

You can drag and drop the files or right click and click 'replace' and then navigate to the files.
If you are having trouble, make sure that 'Edit mode' is turned on towards the top right of the OpenIV window.
I suggest using a mods folder for any vehicle modding

VEHICLES.META
Using OpenIV or similar program, put the file (vehicles.meta) into:
            update>update.rpf>common>data>levels>gta5

If you already have a modifed version of the vehicles.meta from another car mod, then extract your current vehicles.meta, 
open it with some sort of text editting software such as Notepad++ or even Wordpad. Then search for 'riot' until you find 
the correct section and paste everything from the Vehicle line document over the original riot code.

HANDLING.META 
Using OpenIV or similar program, put the file (handling.meta) into:
            update>update.rpf>common>data

If you already have a modifed version of the handling.meta from another car mod, then extract your current handling.meta, 
open it with some sort of text editting software such as Notepad++ or even Wordpad. Then search for 'riot' until you find 
the correct section and paste everything from the Handling line document over the original riot handling.
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
VEHICLELAYOUTS.META 
Using OpenIV or similar program, put the file (handling.meta) into:
            update>update.rpf>common>data>ai

You need this file in order to have the extra seats in the back work properly. It's unlikely you have a modifed version of this file,
but if you do, do the same process stated above. Extract it with OpenIV, edit it with Notepad++ or wordpad, replace the 
riot van lines with the ones from the download vehiclelayouts.meta
